# Master Prompt For Codex

Нужно разработать premium personal portfolio website для видеографа Рустама Бариева.

## Цель
Сайт должен не просто показывать работы, а формировать ощущение:
- профессиональный видеограф полного цикла
- современный workflow
- сильный визуальный вкус
- надёжность
- аккуратная работа с клиентской задачей
- AI как усиление процесса, а не gimmick

## Позиционирование
Рустам Бариев — видеограф полного цикла из Сочи.
Съёмка, монтаж, цветокоррекция, reels, интервью, подкасты, коммерческие и имиджевые видео.
AI используется в postproduction и prep-задачах там, где это реально усиливает результат.

## Технический стек
- Next.js
- React
- Tailwind CSS
- компонентная архитектура
- хорошая семантика
- SEO-friendly базовая структура
- высокая скорость загрузки
- lazy loading для видео
- удобная замена контента через отдельные data/content файлы

## Нужные страницы
1. Главная
2. Кейсы
3. Отдельная страница кейса
4. Контакты

## Главная должна содержать
- hero section с video background или showreel
- trust bar
- services section
- featured cases
- showreel section
- workflow section
- AI section
- about section
- FAQ
- final contact CTA

## Что сделать
- продумать архитектуру проекта
- создать компоненты
- реализовать страницы
- использовать placeholder media и комментарии, где заменять материалы
- сделать аккуратные анимации
- подготовить удобную структуру контента
- предусмотреть лёгкую замену текстов и кейсов
- сделать визуально дорогой, но сдержанный результат
